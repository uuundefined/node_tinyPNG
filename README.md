# tinyPng_2023

单文件两百行代码搞定，暂时不想开源。

# 使用方法

pkg打包WIN版本。使用方法，直接拖动文件夹或者文件到 exe文件上。
就能把整个文件夹下的 jpg jpeg webp 文件全部压缩好。

# 特点

不需要API_KEY，没数量限制
文件夹不限制里面的文件数量，目录结构
可以保持目录结构，递归压缩子文件夹
